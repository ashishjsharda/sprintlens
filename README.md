# SprintLens — Scrum Intelligence for Jira

Gantt charts, stale detection, overallocation alerts and executive dashboards built natively into Jira.

## Features

- Sprint Gantt chart with live timeline and stale color coding
- Stale issue detection with configurable thresholds (3/5/7 days)
- Overallocation alerts for engineers with too many active tasks
- Executive dashboard with velocity and burndown charts
- Per-engineer resource drill-down across 12 sprints
- Slack webhook integration for real-time alerts
- Fully configurable per project

## Install

Available on the Atlassian Marketplace — search "SprintLens"
