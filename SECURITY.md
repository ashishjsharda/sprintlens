# Security Policy — SprintLens

## Overview
SprintLens is a read-only Jira sprint dashboard app built on Atlassian Forge. 
It does not collect, store, or process any end-user data outside of 
Atlassian's platform.

## Data Handling
- No end-user data is stored outside Atlassian apps and services
- No backend servers or external databases are used
- All data is fetched directly from Atlassian's Jira APIs at runtime 
  and displayed in the browser
- No user credentials, tokens, or personal information are collected

## Vulnerability Reporting
If you discover a security vulnerability in SprintLens, please report it by 
emailing: ashishjsharda@gmail.com
We will respond within 48 hours and work to address valid issues promptly.

## Compliance
SprintLens participates in the Atlassian Marketplace Security Bug Bounty Program.
